## Rock Paper Scissors
### Program: 
The purpose of this program is to randomly select a choice of rock, paper, scissors, spock, or lizard to play against the user.

### Process: 
The program randomly selects a random number which is assigned a value of rock, paper, scissors, etc. and the value of user input is compared with the computer input.   

### Results: 
The program outputs whether or not the user wins or loses each round giving the user the option to replay each round.

### Skills used:

  * Random
  * If/else statements
  * tolower() method
  * While loops
